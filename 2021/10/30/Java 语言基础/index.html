内部类

  • 一个内部类只能被其外部类使用

  • 一个内部类被编译为“外部类名$内部类名.class”

  • 内部类可以引用外部类中的数据和方法

  • static 内部类不能访问外部类的非静态成员

  • 在其他类中创建内部类：

匿名内部类

  • 匿名内部类总是从一个父类或实现一个接口
  • 匿名内部类不能有显式 extends 或 implements
  • 匿名内部类必须实现父类或接口所有抽象方法
  • 匿名内部类总是用父类无参构造方法创建实例，接口则是 Object()
  • 匿名内部类被编译成 OuterClassName$n.class
