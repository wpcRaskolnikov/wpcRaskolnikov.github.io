Typora 快捷键

Posted on 2021-12-20 Edited on 2021-10-30

打开文件

操作 快捷键
打开文件 Ctrl + O
打开最近一个关闭的文件 Ctrl + Shift + T
快速打开最近打开过的文件 Ctrl + P

选择文本

操作 快捷键
选中当前词 Ctrl + D/双击鼠标
选中当前句 Ctrl + L/三击鼠标
选中当前格式文本 Ctrl + E

样式

操作 快捷键
斜体 Ctrl + I
粗体 Ctrl + B
超链接 Ctrl + K
下划线 Ctrl + U
删除线 spanlt + Shift + 5
引用 Ctrl + Shift + Q
代码 Ctrl + Shift + K
清除样式 Ctrl + \

查找删除

操作 快捷键
删除当前词 Ctrl + Shift + D
查找上一个 F3
查找下一个 Shift + F3

跳转

操作 快捷键
跳转选中文本 Ctrl + J
跳转文首 Ctrl + Home
跳转文尾 Ctrl + End

标题与表格

操作 快捷键
提升标题等级 Ctrl + =
降低标题等级 Ctrl + -
设为普通文本 Ctrl + 0
设为六级标题 Ctrl + 6
表格增加行 Ctrl + Enter
表格删除行 Ctrl + Shift + Bspanckspspance
